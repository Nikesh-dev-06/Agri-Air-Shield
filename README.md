# 🔥 AgriAirShield - Burn Sentinel

### AI-Powered Crop Burn Prevention & AQI Forecasting System

AgriAirShield is an intelligent web application developed to monitor, predict, and visualize crop residue burning activities and their impact on air quality across agricultural regions.

Built during the **CSI KEC Student Branch Hackathon**, this project combines interactive geospatial visualization, predictive analytics, AQI forecasting, and sustainable agricultural alternatives into a unified platform.

---

# 🚀 Features

## 🌾 Crop Burn Monitoring
- Interactive map for monitoring crop burning hotspots
- Region-based burn visualization
- Intensity categorization:
  - Low
  - Medium
  - High
  - Extreme

## 🤖 AI-Based Burn Prediction
- 14-day burn probability forecasting
- Machine learning prediction models
- Burn intensity forecasting
- Risk score analysis

## 🌬️ AQI Forecasting
- Real-time AQI impact visualization
- Pollution spread prediction
- City-wise AQI risk analysis
- Wind-direction-based impact tracking

## 📊 Data Analytics Dashboard
- Monthly and yearly burn statistics
- Burn event trend visualization
- Fire Radiative Power (FRP) analysis
- High-risk zone identification

## 💰 Sustainable Crop Residue Alternatives
The platform also suggests profitable alternatives to crop burning such as:
- Biochar production
- Biomass energy
- Organic compost
- Bio-briquettes
- Mushroom cultivation
- Ethanol production
- Animal feed pellets
